# Git Fundamentals — Practice Repository

Hands-on practice of core Git workflows using real command-line operations.

---

## Covered Git Concepts

| Area | Commands |
|------|----------|
| Repository setup | `git init` |
| Staging & commits | `git add`, `git commit` |
| File modification | tracked file updates |
| Discarding changes | `git checkout -- file` |
| Unstaging | `git reset file` |
| Safe undo | `git revert` |
| History rewrite | `git reset --hard` |
| Commit editing | `git commit --amend` |
